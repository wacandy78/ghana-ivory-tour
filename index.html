兩大國家公園

莫宗瑞亞國家公園

🦅 多樣化鳥類生態
位於迦納北部薩瓦納地區，莫宗瑞亞是西非最大的國家公園。園內茂密的森林和遼闊的草原為眾多野生動物提供了完美的棲息地。

主要鳥類：

✓ 各種鷹鷲與獵鷹
✓ 鸛鳥和蒼鷺
✓ 五彩斑斕的蜂虎
✓ 珍稀的戰鷲和非洲雕

🏨 Zaina Lodge 🚶 專業導遊 📍 隱蔽觀察點 🌙 夜間探險
體驗亮點： 每天清晨由專業導遊帶領進行深度森林漫步和野生動物觀察。園內設有多個隱蔽的觀鳥點，距離大象和其他野生動物保持安全距離。住在優雅的樹屋，盡享自然與奢華的完美結合。

卡庫姆國家公園

🦅 超過 300 種鳥類
位於迦納中南部的雨林地區，卡庫姆以其令人驚嘆的 7 條懸空走廊聞名世界。走在距離森林地面 27 公尺高的樹頂上，俯瞰這片古老的雨林生態系統。

主要鳥類：

✓ 華麗的翠鳥
✓ 多彩的蜂虎
✓ 稀有的夜鷲
✓ 吸蜜鳥和啄木鳥

🌳 樹冠走廊 🔭 遠距望遠鏡觀察 🌙 夜間生態探險 📸 攝影天堂
體驗亮點： 穿梭於 333 公尺長的懸空走廊，與鳥類在樹冠層相遇。使用高倍望遠鏡從訪客中心觀察稀有鳥種，或參加引人入勝的夜間生態探險。園內設有露營設施和樹屋住宿，深度體驗原始雨林。

延伸探險

從迦納跨越邊界，深入象牙海岸的海灘與城市魅力

Assinie - 象牙海岸海灘度假

🏖️ 沿海生態天堂
Assinie 是象牙海岸東南部的靜謐海濱小鎮，擁有金黃色沙灘和清澈的大西洋海水。這個度假勝地結合了自然美景與文化特色，是逃離城市喧囂的理想之地。

沿海鳥類：

✓ 鸕鷀和蒼鷺群
✓ 魚鷹和海鷲
✓ 翠鳥和潛水鳥
✓ 沙灘鳶（稀有）

🌊 泻湖生態 🐚 沿海濕地 🌅 日出觀賞 🏨 度假村
體驗亮點： 清晨在泻湖觀賞遷徙水鳥，漫步沙灘尋覓沿海物種。下午在海邊度假村放鬆，享受新鮮海鮮與冷飲。Assinie 是從野生動物探險到悠閒度假的完美轉銜。

Abidjan - 象牙海岸現代都市

🏙️ 西非文化樞紐
Abidjan 是象牙海岸最大的城市，也是西非最現代化的都市之一。這座城市坐落在美麗的潟湖之上，融合了當代建築、豐富的文化與充滿活力的夜生活。

城市自然體驗：

✓ Banco 國家公園城市森林
✓ 潟湖水鳥觀察
✓ 市郊自然保護區
✓ 都市公園棲息地

🎨 當代美術館 🛍️ 現代購物中心 🍽️ 美食之都 🌃 夜生活
體驗亮點： 參訪美術館與文化中心探索象牙海岸藝術。在 Banco 國家公園進行城市邊緣賞鳥，發現潟湖邊的野生動物。享受國際美食與購物，感受西非最繁華的都市活力。
